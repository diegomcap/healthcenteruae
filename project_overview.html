Health & Pilates Center - Project Overview

Project Description

This web application serves as a secure portal for Health & Pilates Center clients to access and complete assessment forms. The system uses a key-based authentication mechanism to ensure that only authorized clients can access the forms, and each access key can only be used once.

Technology Stack

HTML5
CSS3
JavaScript
LocalStorage API
Node.js/Express
Python/Flask

The project can be run using either Node.js or Python as the backend server. Both implementations are provided:

  • server.js - Node.js/Express implementation
  • server.py - Python/Flask implementation

Security Implementation

Authentication System

The system uses a key-based authentication mechanism stored in the browser's localStorage. Each access key can only be used once to complete an assessment form.

Key Validation Process:

  1. User enters access key on login page
  2. System checks if key exists in validKeys array
  3. System checks if key has been used before (usedKeys array)
  4. If valid and unused, user is granted access
  5. Key is marked as used after form submission

Server Implementation

Running with Node.js

The server will start on http://localhost:3000

Running with Python

The Flask server will start on http://localhost:3000

Form Submission

Forms are handled by the form_handler.js script which:

  • Intercepts form submissions
  • Sends data via AJAX
  • Marks the access key as used
  • Redirects to the thank you page
